# Códice FP Digital
Proyecto para mostrar con ejemplos las distintas unidades, examenes y materias de Formación Profesional de forma visual y con ejemplos.

## Enfoque en ciclos de Formación Profesional y Grados de especialización de informática.
El enfoque principal es la descomposición modular de los currículos oficiales (ASIR, DAM, DAW). Cada sección está diseñada como una unidad de aprendizaje independiente que conecta lenguajes de marcas, lógica de programación y gestión de sistemas con ejemplos interactivos, facilitando la transición entre la teoría del examen y la práctica profesional.
